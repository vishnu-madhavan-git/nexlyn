PROJECT SUMMARY
================================================================================

WHAT IS IT?
A React + TypeScript single-page application that serves as an AI-powered
distribution platform for MikroTik networking hardware. Integrates Google's
Gemini AI for intelligent technical support.

TECH STACK:
- React 19.2.3 + TypeScript
- Vite 6.2.0 (build tool)
- Tailwind CSS (styling)
- Google Gemini AI (chatbot)
- Cloudinary (image hosting)
- Recharts (analytics)

KEY FEATURES:
- AI chatbot with streaming responses
- Product catalog with filtering
- Admin panel with CRUD operations
- WhatsApp integration for B2B
- Dark/Light theme toggle
- Fully responsive design

CODEBASE SIZE:
- App.tsx: ~2,000 lines (main component)
- constants.tsx: ~400 lines (data)
- Services: ~100 lines (AI + image upload)
- Total: ~2,500 lines of code

================================================================================
QUICK START
================================================================================

# Clone
git clone https://github.com/vishnu-madhavan-git/NEXLYN---v2.git
cd NEXLYN---v2

# Install
npm install

# Configure environment (.env.local)
GEMINI_API_KEY=your_key
VITE_CLOUDINARY_CLOUD_NAME=your_cloud
VITE_CLOUDINARY_UPLOAD_PRESET=your_preset

# Run
npm run dev

# Build
npm run build
